What is PDXShelter?

PDXShelter is a resource for both providers and tenants of homeless shelters. Users can view shelter bed openings near them in real time and shelters can update their count on the fly. Built to be used on both web and mobile platforms, we hope to help keep a roof over heads to those in need.

Shelter Info

PDX Shelter is a Portland-based online resource for homeless shelters and the homeless in Portland. Users can view bed availability, create a user profile and make shelter reservations. The mobile and web app allows shelters to update their bed availability and check in guests while saving the information to a database for quick future check-in.
